## About
DBCMDPP (Discord Bot Command Plus Plus) is a Discord bot written in C++ using the [D++ (DPP)](https://github.com/brainboxdotcc/DPP) library.

## Requirements

**Windows**
- MSVC (Visual Studio 2022 recommended)
- CMake 3.31+

**macOS**
- Homebrew
- CMake 3.31+

## Quick Start

**Windows**
```bash
cmake -B build
cmake --build build --config Debug
```

**macOS**
```bash
brew install dpp
cmake -B build
cmake --build build
```
